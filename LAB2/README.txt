Para compilar el codigo asegurarse de:
	- Tener la carpeta src con el codigo fuente
	- Tener la carpeta incl con los .h
	- Tener el makefile proporcionado
	- Una imagen cuadrada .bmp

Para la compilacion ejecutar:
	make

Para la utilizacion utilizar este comando de ejemplo:
	./lab2 -i imagen.bmp -o histograma.txt -L 2 -B 256

	"ejecutable" -i "entrada" -o "salida" -L "nivel" -B "bins"
